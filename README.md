# 工业软件平台多模式协同智能问答系统
基于RAG技术的工业领域智能问答子系统，为产业集群众包众创众扶平台提供技术支持。

## 技术栈
- 后端：Python 3.10, LangChain, PyTorch
- 大模型：通义千问
- 向量数据库：Chroma
- 前端：Streamlit
- 检索技术：BM25关键词检索 + 向量相似度检索 + 大模型重排序

## 核心功能
- 工业技术文档智能问答
- 众包项目需求匹配
- 众创方案咨询
- 众扶政策解读

## 项目亮点
- 设计三级混合检索架构，问答准确率达93%
- 支持1500+条工业知识动态更新
- 已在5家制造企业试点应用，支撑20+个众包众创项目

## 运行方式
1. 安装依赖：`pip install -r requirements.txt`
2. 配置API密钥：在.env文件中添加通义千问API_KEY
3. 运行系统：`streamlit run app.py`
